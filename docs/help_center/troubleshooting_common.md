# Common Troubleshooting Steps

## General Issues

### App Not Loading
1. Check internet connection
2. Clear browser cache
3. Disable browser extensions
4. Try incognito/private mode
5. Try different browser
6. Check status.example.com

### Slow Performance
1. Close unused tabs/apps
2. Clear browser cache
3. Check network speed
4. Disable complex embeds
5. Try desktop app

### Sync Issues
1. Check internet connection
2. Wait a few minutes
3. Refresh page (Cmd/Ctrl + R)
4. Log out and back in
5. Check for conflicts

## Browser-Specific

### Chrome
- Clear cache: Settings → Privacy → Clear data
- Disable extensions one by one
- Update Chrome

### Safari
- Enable JavaScript
- Disable content blockers
- Update Safari

### Firefox
- Disable enhanced tracking
- Clear cookies for site
- Update Firefox

## Mobile App Issues

### App Crashing
1. Force close app
2. Restart device
3. Update app
4. Reinstall app
5. Check device storage

### Sync Not Working
1. Check internet connection
2. Pull to refresh
3. Log out and back in
4. Check background refresh settings

## Reporting Bugs

Include in your report:
- What you expected
- What happened instead
- Steps to reproduce
- Browser/device info
- Screenshots if possible
- Console errors (for developers)
