# Feature Requests and Feedback

## How to Submit Feature Requests

1. Go to Help → Feature Requests
2. Search existing requests first
3. Vote on similar requests
4. Or submit new request with details

## What to Include

- Clear description of the feature
- Your use case / problem it solves
- How you currently work around it
- Priority for your workflow

## Request Process

1. **Submitted**: Under review
2. **Under Consideration**: Being evaluated
3. **Planned**: On our roadmap
4. **In Progress**: Being built
5. **Shipped**: Available!

## Popular Requested Features

We share top-voted features monthly:
- Check our public roadmap
- Join beta programs early
- Subscribe to changelog

## Feedback Channels

- In-app feedback button
- Community forum
- Twitter @example_app
- Email feedback@example.com

## Bug Reports vs Feature Requests

**Bug Report:**
- Something broken/not working
- Unexpected behavior
- Error messages

**Feature Request:**
- New functionality
- Improvement to existing feature
- Integration requests
