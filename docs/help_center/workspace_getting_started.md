# Getting Started with Workspaces

## What is a Workspace?

A workspace is your team's shared environment. It contains all pages, databases, and settings for your team.

## Creating a Workspace

1. Click "Create or Join Workspace" from sidebar
2. Choose "Create a Workspace"
3. Enter workspace name
4. Invite team members (optional)
5. Select a plan

## Joining an Existing Workspace

**Via Invitation:**
1. Accept email invitation
2. Click link to join
3. Complete profile setup

**Via Domain:**
1. If workspace uses SSO with verified domain
2. Sign in with company email
3. Auto-join enabled workspaces

## Workspace Settings

Access via Settings & Members:
- **General**: Name, icon, description
- **Members**: Invite, manage roles
- **Security**: SSO, 2FA requirements
- **Billing**: Plan, invoices
- **Integrations**: Connected apps

## Switching Workspaces

- Click workspace icon in sidebar
- Select from list of workspaces
- Or use keyboard shortcut: Cmd/Ctrl + Shift + O

## Troubleshooting

**Can't see workspace:**
- Check you're logged into correct account
- Verify invitation was accepted
- Contact workspace admin

**Can't create workspace:**
- Check email is verified
- Try different browser
- Clear cache and cookies
