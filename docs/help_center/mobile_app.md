# Mobile App Guide

## Getting Started

### Download
- iOS: App Store (iOS 14+)
- Android: Google Play (Android 8+)

### Sign In
1. Open app
2. Enter email
3. Check email for magic link
4. Tap link to sign in

Or use SSO if configured.

## Features

### Available on Mobile
- View and edit pages
- Quick capture notes
- Search content
- Comment on pages
- Receive notifications
- Offline access (limited)

### Not Yet Available
- Dark mode
- Advanced formatting
- Database views (limited)
- Templates
- Page analytics

## Syncing

### Automatic Sync
- Changes sync when online
- Sync indicator in top right
- Pull down to force sync

### Offline Mode
- Recent pages cached
- Edits saved locally
- Sync when reconnected
- Conflicts flagged

## Troubleshooting

### App Crashes
1. Update to latest version
2. Clear app cache
3. Reinstall if needed
4. Report persistent issues

### Sync Issues
1. Check internet connection
2. Force sync (pull down)
3. Sign out and back in
4. Check server status

### Login Problems
1. Request new magic link
2. Check spam folder
3. Use SSO if available
4. Contact support

## Tips

- Enable push notifications
- Use quick capture widget
- Star important pages
- Enable biometric unlock
