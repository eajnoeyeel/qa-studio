# Keyboard Shortcuts

## Navigation

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Search | ⌘ + K | Ctrl + K |
| Quick find | ⌘ + P | Ctrl + P |
| Go back | ⌘ + [ | Alt + ← |
| Go forward | ⌘ + ] | Alt + → |
| Sidebar toggle | ⌘ + \ | Ctrl + \ |

## Content Creation

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| New page | ⌘ + N | Ctrl + N |
| New page in same location | ⌘ + Shift + N | Ctrl + Shift + N |
| Duplicate page | ⌘ + D | Ctrl + D |
| Delete page | ⌘ + Shift + D | Ctrl + Shift + D |

## Editing

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Bold | ⌘ + B | Ctrl + B |
| Italic | ⌘ + I | Ctrl + I |
| Underline | ⌘ + U | Ctrl + U |
| Strikethrough | ⌘ + Shift + S | Ctrl + Shift + S |
| Code | ⌘ + E | Ctrl + E |
| Link | ⌘ + K | Ctrl + K |

## Blocks

| Action | Shortcut |
|--------|----------|
| Heading 1 | /h1 or # |
| Heading 2 | /h2 or ## |
| Heading 3 | /h3 or ### |
| Bullet list | /bullet or - |
| Numbered list | /number or 1. |
| Todo | /todo or [] |
| Code block | /code or ``` |
| Quote | /quote or > |
| Divider | /divider or --- |

## Selection

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Select all | ⌘ + A | Ctrl + A |
| Select block | Esc then ↑/↓ | Esc then ↑/↓ |
| Multi-select | Shift + Click | Shift + Click |
| Extend selection | Shift + ↑/↓ | Shift + ↑/↓ |

## Comments

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Add comment | ⌘ + Shift + M | Ctrl + Shift + M |
| View comments | ⌘ + Shift + C | Ctrl + Shift + C |

## View Complete List

Press ⌘ + / (Mac) or Ctrl + / (Windows) in app
