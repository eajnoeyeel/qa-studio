# Import and Export Data

## Supported Import Formats

- Markdown (.md)
- HTML files
- CSV (for databases)
- Notion export
- Evernote (.enex)
- Confluence (via API)
- Trello (via API)
- Google Docs (limited)

## Importing Content

### From Files
1. Click "Import" in sidebar
2. Select file type
3. Upload file(s)
4. Choose destination workspace
5. Review and confirm

### From Other Apps
1. Go to Settings → Import
2. Select source application
3. Authorize connection
4. Select what to import
5. Map to workspace structure

## Export Options

### Single Page
1. Open page menu (•••)
2. Click "Export"
3. Choose format (Markdown, PDF, HTML)
4. Download file

### Entire Workspace
1. Go to Settings → Export
2. Choose format
3. Select what to include
4. Request export
5. Download link via email (large exports)

## Sync Integrations

### Connected Apps
- Slack: Share pages, get notifications
- GitHub: Embed issues, PRs
- Figma: Embed designs
- Google Drive: Link documents

### API Access
- Full API for custom integrations
- Rate limits apply per plan
- OAuth 2.0 authentication

## Troubleshooting

**Import fails:**
- Check file size (max 50MB per file)
- Verify format matches selection
- Try smaller batch

**Missing content after import:**
- Some formatting may not transfer
- Check import log for errors
- Manual cleanup may be needed

**Export taking too long:**
- Large workspaces take time
- You'll receive email when ready
- Link valid for 7 days
