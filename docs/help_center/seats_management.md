# Managing Team Seats and Licenses

## Understanding Seats

A seat is a license for one user to access your workspace with full member permissions.

### What counts as a seat?
- Active workspace members
- Pending invitations (until declined/expired)
- NOT guests (separate limit)

### What doesn't count?
- Guest users (up to 5 free, then $5/guest)
- Deactivated members
- Single sign-on user provisioning (until they log in)

## Adding Seats

### Manual Addition
1. Go to Settings → Members
2. Click "Invite Members"
3. Enter email addresses
4. Confirm seat addition if over limit

### Automatic Addition
- Inviting members auto-adds seats if needed
- Prorated charge for current billing cycle
- Example: Add 1 seat mid-month = half price that month

## Removing Seats

### Remove Members First
1. Go to Settings → Members
2. Find member to remove
3. Click "..." → Remove from workspace
4. Member loses access immediately

### Reduce Seat Count
1. Go to Settings → Billing
2. Click "Manage Seats"
3. Set new seat count
4. Takes effect next billing cycle

**Note**: Cannot reduce below current member count.

## Best Practices

- Audit unused seats quarterly
- Use SSO auto-provisioning for large teams
- Convert inactive members to guests when possible
- Set up seat alerts before limit reached
