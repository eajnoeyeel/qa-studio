# Classification Task Evaluation Guide

## What to Look For

- Clear identification of available categories
- Explicit selection with justification
- Consideration of ambiguous cases
- Confidence indication where appropriate

## Common Failure Patterns

- **Instruction Miss**: Classifying into a category not in the provided options
- **Incomplete Answer**: Choosing a label without justification
- **Logic Error**: Contradictory reasoning for the chosen category
- **Partial Answer**: Only classifying some items in a multi-item task

## Scoring Guidelines

### Instruction Following
- 5: Correct category selected, format matches, all items classified
- 3: Correct category but missing required justification
- 1: Wrong category or ignores the classification task

### Reasoning Quality
- 5: Clear criteria-based justification with evidence from the text
- 3: Reasonable choice with basic justification
- 1: No justification or contradictory reasoning

### Completeness
- 5: All items classified, edge cases discussed, confidence noted
- 3: Main classification done but missing secondary analysis
- 1: Only partial classification attempted

### Clarity
- 5: Category clearly stated, justification well-organized
- 3: Answer identifiable but could be structured better
- 1: Unclear which category was chosen
