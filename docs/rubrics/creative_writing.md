# Creative Writing Task Evaluation Guide

## What to Look For

- Adherence to specified genre, tone, and constraints
- Incorporation of all prompt elements (characters, setting, theme)
- Consistent voice and style throughout
- Narrative structure with beginning, middle, and end

## Common Failure Patterns

- **Instruction Miss**: Ignoring genre, tone, or specific prompt requirements
- **Incomplete Answer**: Story that ends abruptly or lacks resolution
- **Format Violation**: Wrong form (e.g. prose when poem requested)
- **Off Topic**: Story that drifts away from the given prompt

## Scoring Guidelines

### Instruction Following
- 5: Every prompt element incorporated, genre and constraints respected
- 3: Main prompt addressed but some elements missing
- 1: Ignores the prompt or writes in wrong genre/format

### Reasoning Quality
- 5: Thoughtful narrative choices, coherent plot, purposeful structure
- 3: Reasonable story but predictable or formulaic
- 1: No discernible narrative logic or plot coherence

### Completeness
- 5: Full narrative arc, all prompt elements woven in naturally
- 3: Has a beginning and end but feels rushed or underdeveloped
- 1: Fragment or severely truncated

### Clarity
- 5: Engaging prose, vivid language, polished writing
- 3: Readable but unremarkable writing
- 1: Poorly written, confusing, or grammatically broken
